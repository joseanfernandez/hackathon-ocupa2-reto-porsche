# TensorFlow.js | Mobilenet

Modelo usado
[aqui](https://github.com/fchollet/deep-learning-models/releases/download/v0.6/mobilenet_2_5_224_tf.h5)
Keras pre-entrenado, modelo mobilenet.

## Instalación:
```Bash
npm install
```
```Bash
yarn install
```

## Ejecución:

Desarrollo:
```Bash
$ npm run watch
```
Producción
```Bash
$ npm run build
```

... y corre en puerto: http://localhost:1234/

Para convertir a JSON onvertir script [here](https://github.com/tensorflow/tfjs-converter).
